# @concordia-protocol/sdk

TypeScript reference implementation of the Concordia Protocol -- signed
agreement primitives for autonomous agents.

Status: alpha. Currently ships the canonical JSON serializer with byte-level
parity against the Python reference implementation. Full primitive set
(mandate, predicate, attestation, session-receipt) ships in subsequent
alpha releases.

Apache-2.0. Spec at https://github.com/eriknewton/concordia-protocol.
